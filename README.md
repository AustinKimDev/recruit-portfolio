# Jidong Kim Portfolio

Cyber-themed portfolio and resume site built with Next.js, React, MapLibre GL, Three.js, and a small local i18n layer.

## Features

- Korean and English content with a top-level language switch
- Dark and light theme support
- Three.js orbital hero visualization
- Interactive MapLibre project atlas
- Project case studies focused on decision, implementation, and outcome
- Resume route at `/resume`

## Stack

- Next.js App Router
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- MapLibre GL
- Three.js

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Checks

npm run lint
